# SDK 方法级索引

## 入口
- 按方法名：`methods-by-name.md`
- 高频方法：`methods-hot.md`
- 生命周期方法：`methods-lifecycle.md`
- 结构化清单：`../manifests/methods.json`

## 结构化字段
- `module`
- `file`
- `package`
- `namespace`
- `class_name`
- `method_name`
- `signature`
- `parameters`
- `return_type`
- `summary`
- `keywords`
- `scenarios`

## 使用规则
- 已知方法名时先看 `methods-by-name.md`。
- 已知生命周期或事件时先看 `methods-lifecycle.md`。
- 只知道业务场景时先看 `scenario-index.md` 和 `keyword-index.md`。
- 找到候选方法后，继续回到类卡片或当前项目依赖确认参数和返回值。
