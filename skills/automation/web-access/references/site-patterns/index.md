# 站点经验记录入口

这个目录只放稳定、可复用、长期有效的站点经验，不记录一次性的排障过程。

## 记录模板
- 站点/产品名
- 触发场景
- 推荐执行面：搜索 / 宿主浏览器 / CDP
- 稳定选择器或目标定位线索
- 常见拦截：登录、验证码、文件上传、弹窗、企业安全
- 安全边界：哪些动作必须让用户确认

## 使用规则
- 只有当站点行为重复出现且足够稳定时才补记录。
- SKILL.md 只放入口；具体站点规则单独成文件，避免把主 skill 写胖。
- 引用站点经验前，先确认当前页面版本和当前宿主能力没有变化。

## 当前模式
- `research-matrix.md`：最新事实核验、多来源交叉验证
- `search-backends.md`：直接来源、宿主搜索和 Brave 后端的取舍
- `browser-interaction.md`：浏览器交互探索阶段的单步动作循环
- `browser-search-fallback.md`：宿主搜索不可用或不稳定时的回退路径
- `auth-session.md`：登录态和会话页面
- `dynamic-spa.md`：单页应用和动态渲染读取
- `dynamic-scraping.md`：稳定重复抓取、分页滚动和批量采集
- `kingdee-vip.md`：金蝶云社区 / VIP 知识页的登录态复用、只读验证与新 tab 继承登录态经验
