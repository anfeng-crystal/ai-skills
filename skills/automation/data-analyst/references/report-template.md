# Report Template

## 简版结构
1. 结论
2. 口径
3. 关键发现
4. 风险与限制
5. 下一步建议

## 可复用句式
- 结论：
  在给定时间范围和样本口径下，`指标` 的主要变化来自 `维度/分组`。
- 口径：
  本次统计时间范围为 `...`，样本范围为 `...`，排除了 `...`。
- 关键发现：
  `维度 A` 对整体结果贡献最高，`维度 B` 是主要波动来源。
- 风险：
  当前数据存在 `空值/重复/口径不清/样本不足`，因此结论应按保守口径理解。
- 建议：
  下一步优先补充 `字段/明细/时间窗口`，或继续下钻 `某个异常维度`。

## 工作簿交付结构
- `Summary`：结论和 KPI
- `Data`：清洗后的明细或汇总
- `Charts`：图表区
- `Notes`：口径、来源和限制
